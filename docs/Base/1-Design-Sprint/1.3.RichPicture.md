# Rich Picture: Carona Amiga - FCTE 

## Introdução

O Rich Picture é uma técnica de modelagem que permite representar visualmente um sistema complexo de forma holística, destacando seus componentes, relacionamentos e interações. O Carona Amiga FCTE é um aplicativo que permite aos estudantes da FCTE/UnB compartilharem caronas de forma segura e organizada.
O Rich Picture mostra os usuários interagindo com o sistema, incluindo estudantes motoristas e passageiros, além de evidenciar fatores externos relacionados ao sistema, como a validação institucional (e-mail UnB), questões de segurança, e o uso de tecnologias como GPS e avaliação de usuários.
Ele fornece uma visão abrangente do funcionamento do aplicativo, destacando fluxos de informação, interações entre os diferentes atores e os principais processos envolvidos, como o pareamento de rotas, definição de pontos de encontro e mecanismos de confiança.

## Metodologia e Ferramentas

Para a elaboração do Rich Picture, foi utilizada a ferramenta Lucidchart, que oferece recursos para criação de diagramas e representações visuais. A construção do diagrama seguiu as seguintes etapas:
- Identificação dos principais atores envolvidos no sistema
- Mapeamento dos processos e fluxos de informação
- Representação das interações entre os diferentes componentes
- Identificação de fatores externos que influenciam o sistema
